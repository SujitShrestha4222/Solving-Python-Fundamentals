Check whether a number is an Armstrong number using a loop.
    An Armstrong number (also called a narcissistic number) is a number that is equal to the 
    sum of its own digits, each raised to the power of the number of digits.
    In simple 👇:
    number: 153
    Here,
    Number of digits = 3
    Digits: 1, 5, 3

    Now raise each digit to power 3 and add:
        =153
        =1³ + 5³ + 3³
        = 1 + 125 + 27
        =153
